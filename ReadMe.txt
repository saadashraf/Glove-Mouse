Run the "Gyromouse.exe" file to run the application.
(It could also be accessed from Codes\Driver\GyroMouse\GyroMouse\bin\Debug)

In order to compile the front end dotnet codes, open the solution file and run.

In order to compile Arduino code, open it in arduino IDE, insert arduino, select board type to Nano, 
processor  to atmega328p, boot loader to atmega328p(old) and then upload the sketch. 
N.B. i2cdev and mpu6050 library must be pre-installed to compile and upload the arduino sketch.
